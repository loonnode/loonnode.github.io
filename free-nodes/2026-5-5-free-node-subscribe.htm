什么是 Shadowrocket？

Shadowrocket 是一款专为 iOS 用户打造的网络代理客户端，最初用于连接 Shadowsocks 等轻量级代理协议，如今已经拓展支持 HTTP、HTTPS、SOCKS5、Vmess（需手动导入配置）等多种协议。

它的核心功能包括：

  • 管理并分流 App 的网络请求；

  • 支持多协议代理（Shadowsocks、Vmess、HTTP 等）；

  • 提供规则自定义能力；

  • 支持速度测速、日志查看、DNS配置等进阶功能；

  • 高度兼容各种配置文件与订阅服务。

更重要的是，它不提供代理节点本身，而是作为代理协议的客户端存在，这意味着你需要单独配置订阅地址或节点信息。

Shadowrocket 的价格是多少？

定价机制与价格区间

价格浮动因素

尽管价格本身固定，但用户在购买过程中可能遇到以下价格影响因素：

  1. 地区差异：部分 App Store 区域如中国大陆无法直接搜索或购买 Shadowrocket，需要切换至美国、香港或其他区域 Apple ID。

  2. Apple ID 代购服务：对于无法自己切换 App Store 的用户，常通过淘宝、QQ群等平台购买美区账号代购，价格会在 20～60 元不等，取决于代购方是否包含美区账号、充值服务等。

  3. 节日促销：虽然官方极少降价，但部分代购商在双11、黑五、618 等节日会推出折扣套餐。

Q3：购买后还需要订阅服务吗？

Q4：会不会被封号或者违法？

Q5：学生适合用 Shadowrocket 吗？

FAQ

PassWall2 如何结合延迟选择最优节点？
PassWall2 定期测速节点延迟，并结合策略组自动选择延迟最低的节点。无需手动操作即可保证网页、视频和游戏流量快速稳定访问，提高整体网络体验。
Hysteria2 协议的主要特点是什么？
Hysteria2 是新一代基于 QUIC 的高速代理协议，支持自适应拥塞控制和零 RTT 连接。相比传统 TCP 协议，它能在高延迟或丢包环境下保持稳定速度，特别适合跨境访问或实时传输场景。
如何在 Windows 上安装 Clash for Windows？
在 Windows 上安装 Clash for Windows 十分简单。只需前往官方 GitHub 页面下载最新版安装包，运行安装程序后导入订阅链接或配置文件即可。安装完成后，用户可以通过图形化界面方便地切换节点、编辑规则和控制代理模式。
如何在 Android 手机上使用 Clash for Android？
Clash for Android 是专为移动设备开发的客户端，安装后导入订阅链接即可使用。用户可设置代理模式（全局、规则、自定义）并查看实时流量信息。它的电量消耗较低，适合长期后台运行。
WinXray 多策略组如何管理大量节点？
多策略组可将节点按地区、延迟或用途分类，并结合规则文件自动分流流量。这样可快速选择最优节点，提高访问速度和稳定性，同时优化节点管理和维护效率。
Clash 如何实现域名白名单策略？
Clash 可以通过策略组结合规则文件，将指定域名或网站加入白名单，实现直连而不经过代理节点。这样可以保证国内访问速度，同时节省节点带宽资源，提高整体网络效率。
PassWall2 是什么软件？
PassWall2 是一个运行在 OpenWRT 路由器上的插件，用于代理流量分流和节点管理。它支持多种协议，包括 V2Ray、Trojan、SSR 等。通过 Web 界面，用户可以灵活设置不同设备走不同代理，适合家庭或小型局域网使用。
V2Ray 中的 alterId 是什么，现在还有必要配置吗？
alterId 曾用于 VMess 的用户混淆机制以防止重放攻击，但在 V2Ray 新版本与 VLess 出现后，alterId 已不再推荐使用。现代配置多使用 UUID 或直接迁移至 VLess/XTLS 以获得更好性能与安全性。
Sing-box 如何兼容 Clash 的配置文件？
Sing-box 支持 Clash 规则格式的解析，用户可直接导入 Clash 的 YAML 配置文件进行使用。虽然部分高级参数需要调整，但大多数分流和策略配置都能自动识别，便于迁移。
VLess 为什么被认为是未来趋势？
VLess 去掉了内置加密层，改为依赖外部 TLS 进行安全通信，这使其更轻量化且更灵活。它能与 Hysteria、Reality 等现代传输方式结合使用，适应性更强，是取代 Vmess 的新标准。
